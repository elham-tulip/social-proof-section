# Social Proof Section

This is a beginner-friendly **responsive Social Proof Section** built using HTML and CSS.  
It is a solution to a Frontend Mentor challenge.

---

## 📌 Overview

This project displays customer testimonials and ratings in a clean, modern layout.  
It is fully responsive and works on both mobile and desktop screens.

---

## 🚀 Features

- Responsive design (mobile & desktop)
- Customer testimonials section
- Star ratings display
- Clean and modern UI
- Beginner-friendly HTML & CSS structure

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Media Queries

---

## 📱 Responsive Design

- Mobile: 375px
- Desktop: 1440px

The layout automatically adjusts depending on screen size.
